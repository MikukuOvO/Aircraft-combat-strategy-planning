# 武汉大学2021级机器学习大作业

## 简介
本项目实现了一个飞机作战策略的规划程序，通过输入测试地图可以自动化的输出飞机的执行策略。

## 项目结构
```
>
    > ./input 输入数据
    > ./output 输出的运行指令
    > ./report 测试结果
    main.cpp 主程序
    Makefile
```

## 运行
在命令行中运行 `Makefile` 文件，命令如下：
```bash
make run TESTCASE=x
```

## 测试
在命令行中运行 `makefile` 文件，命令如下：
```bash
make check TESTCASE=x
```
